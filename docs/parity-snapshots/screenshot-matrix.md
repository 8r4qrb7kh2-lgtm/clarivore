## Capture requirements

- Use `editorParity=legacy` in URL.
- For editor captures, use manager account with edit access.
- For mobile captures, use viewport width <= 430 px.
- For multipage captures, ensure at least 3 menu images and page-switch controls visible.
- Capture full-page screenshots for parity comparisons.
